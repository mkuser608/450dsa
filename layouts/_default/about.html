How to Use

  1. Browse Topics: Start by exploring the different DSA topics on the home page.
  2. Select a Topic: Click on a topic to see all the problems in that category.
  3. Solve Problems: Click on the problem links to access the problem on platforms like GeeksForGeeks or Coding Ninjas.
  4. Track Progress: After solving a problem, mark it as completed by checking the box next to it.
  5. Add Notes: Click the notes icon to add personal notes about your approach or things to remember.
  6. Bookmark: Use the bookmark icon to save important problems for later review.
  7. Create Account (Optional): Sign in with Google or GitHub to sync your progress across devices and appear on the leaderboard.
